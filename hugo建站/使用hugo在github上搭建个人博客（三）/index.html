使用hugo在github搭建个人博客（三）

Thu, Sep 3, 2015 in using tags

博客的部署与更新

git安装与教程

廖雪峰的git教程，通俗易懂 www.liaoxuefeng.com/ 要完成下面的操作只需要看 安装git、创建版本库、远程仓库这几部分即可。

生成 public 目录

在 Github 上创建一个 用户名.github.io 的 Repository。

在本地生成public，如果之前已经生成过，记得要先删除。

hugo --baseUrl="http://zwenjiao.github.io/

推送到github远程仓库

$ cd public 
$ git init  
$ git remote add origin https://github.com/zwenjiao/zwenjiao.github.io.git // 添加远程仓库
$ git add -A  // 跟踪所有文件
$ git commit -m "first commit" // 提交
$ git push -u origin master // 推送到远程仓库

注意：

git使用ssh加密的，remote语句改为

$ git remote add origin git@github.com：zwenjiao/zwenjiao.github.io.git

push出错时语句改为

$ git push -u origin master -f  //强制覆盖

此时访问 用户名.github.io就可以在线查看你的网站了。

手机otg线制作教程

Thu, Sep 11, 2014 in using tags

OTG数据线与普通USB数据线的区别：

USB数据线已是日常生活中司空见惯的必备数码配件，然而随着手机行业的快速发展，OTG线也逐渐进入大众的视野。OTG线跟USB线长得如此相似，那么它们又有什么区别呢？

简单地说，有了OTG数据线，你的手机就无需经过PC或笔记本，可直接通过OTG数据线与U盘、鼠标、键盘等USB设备连接，也就是说手机可直接接U盘查找资料、接鼠标玩游戏、接键盘快速打字、接相机浏览图片，即拍即传。换句话说，OTG线省去了各个设备必须经过电脑才能互通信息的步骤，既扩展了手机功能，也让我们更便利地与互联网相连，更适应这个移动互联时代的发展。平板等其他带有OTG功能的数码设备也同样适用。当然，前提是你的数码设备得支持OTG功能哦！

OTG线与普通USB线的区别还在于：USB数据线连接电脑的时候，外接设备由电脑供电，而OTG数据线连接U盘等设备的时候，手机等主体对象必须给U盘等外设供电。

那么手机是如何识别USB数据线和OTG数据线的呢？拆解两根线之后，你会发现USB数据线的第4pin是悬空的，OTG数据线的第4pin是短接到地的，手机芯片应该是通过第4pin判断插入的是OTG数据线还是USB数据线，并以此来决定是否打开OTG供电功能。

传统 usb 接口处pin 有： 1. vbus (5v)2. GND3. D+4. D-

OTG线只有Micro接口, 总共出5pin1. vbus2. GND3. D+4. D-5. ID

USB线材里面跑的都是4条线，区别在于接头。手机/平板都使用Micro USB接头, 接普通USB线，ID pin悬空， 所以运作在 client/peripheral mode。如果把ID GND接地，就会进入 host mode。

4.连接手机的USB小头的焊接点可能会有所不同。 自己就发现两种，一种是一排4个焊点，一种是两排5个焊点。 但其实连接手机的USB小头触点都是有5个的。

5.一排4个焊点这种接线有些麻烦，可以把接头上面的铁壳拆下， 在触点中间位置点焊一下短接GND和ID触点，注意不要把其它触点也短接了。

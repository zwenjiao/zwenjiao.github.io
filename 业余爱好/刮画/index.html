刮画

Mon, Jun 2, 2014 in using tags

最近的一些刮画作品！

在贴吧里面看到的刮画，于是买回来准备试一下，第一幅是许嵩，在网上找了一张图片，然后用ps处理成黑白的，最后刮出来觉得不太像，像汪苏泷，脸刮肥了！

第二张是一棵树，随意刮得，沿着树的枝条走向……

下面这一幅，早上起床准备写代码，随手拿起身边的圆规，画了一些圆，然后灵感就来了，于是画了很多奇奇怪怪的眼睛，好多自然界不存在的眼睛~

我把这幅画叫做《眼》

喜欢这种简单的立体几何无限循环的感觉，会产生很奇妙的效果，大自然真的很神奇！

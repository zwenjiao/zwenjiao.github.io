PS转手绘

Fri, Jul 11, 2014 in using tags

素材还是许嵩，谁让我唯一追的星就是他呢！

找了一张高清桌面，第一次弄，不停地试效果，弄了一个下午，当然P自己喜欢的不会觉得累喽！

不过学到很多新的PS技能！赞赞赞！

PS：

跟人像修图差不多，不过就是美化程度很大，往假里做就行！

1、脸加了高斯模糊，又加了一层白色图层，降低透明度。

2、没有手绘板，头发太难画，用涂抹工具涂抹了三次，沿着头发走向，每次的画笔直径都变小。

3、脸部用加深减淡工具刻画明暗对比，使五官更立体。

4、嘴唇涂上别的颜色，混合模式改为柔光。

5、用钢笔画眼睫毛，画笔描边。

6、在鼻头和嘴唇上面点上白点当高光。

7、用钢笔工具勾出整体轮廓，用画笔描边。

8、将花朵用涂抹工具沿花瓣和叶子走向涂抹。

9、最后加一点对比度，曲线提亮，用别的画笔点缀背景。

ok！

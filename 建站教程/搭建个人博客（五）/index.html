搭建个人博客（五）

Sun, Mar 15, 2015 in using tags

前面我们说的建站方法，虽然刚开始比较简单，可是后期还需要很多修改，在本地搭建测试环境也十分麻烦，那么有没有什么更简单的方法？

我们介绍用静态生成器在github上建造免费站点:

github为程序员提供了免费源代码托管空间，允许大家在GitHub上创建自己的博客网站或主页（github pages），而且免费，不限流量，还可以绑定自己的域名。github只支持上传静态的网页。

所以我们就可以利用静态生成器生成静态页面，然后上传到github，将guthub作为一个免费的，不限流量的空间。

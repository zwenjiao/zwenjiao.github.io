搭建个人博客（三）

Sun, Mar 15, 2015 in using tags

现在我们概念已经了解的差不多了，动手来建一个网站吧。 现在很多的空间提供商有一键建站的功能，常见的wordpress等都可以直接一键部署环境，然后将下载好的wordpress压缩包上传上去就可以了！

部署好之后如果对界面主题不满意可以更换自带的主题， 想自己捣鼓的同学也可以在本地搭建测试环境，自己写界面，在本地测试好之后再上传到空间，

搭建本地测试环境可以下载php集成环境安装包，appserv，xampp等，下载好之后所有需要的环境和工具就都有了，不需要一个一个下载安装，比如 AppServ 所包含的软件有：Apache、Apache Monitor、PHP、MySQL、phpMyAdmin等。
